# 俄罗斯方块游戏 - 技术规范

## 1. 项目概述
- **项目名称**: tetris-game
- **项目类型**: SpringBoot Web游戏应用
- **核心功能**: 经典俄罗斯方块游戏，支持键盘控制、分数计算、游戏排名
- **目标用户**: 休闲游戏玩家

## 2. 技术栈
- Java 8
- SpringBoot 2.7.x
- Maven
- HTML5 Canvas + JavaScript (前端)
- Thymeleaf (模板引擎)

## 3. 游戏功能

### 3.1 方块类型 (7种)
- I型 (青色)
- O型 (黄色)
- T型 (紫色)
- S型 (绿色)
- Z型 (红色)
- J型 (蓝色)
- L型 (橙色)

### 3.2 游戏机制
- 10列 x 20行游戏区域
- 方块自动下落 (初始速度 1格/秒)
- 速度随等级提升而加快
- 消除1行得分100，2行300，3行500，4行800
- 等级 = 总分 / 1000 + 1
- 游戏结束条件：方块堆积到顶部

### 3.3 控制方式
- ← → : 左右移动
- ↓ : 加速下落
- ↑ / W : 顺时针旋转
- 空格 : 硬降 (直接落到底部)
- P : 暂停/继续
- R : 重新开始

### 3.4 额外功能
- 最高分记录 (本地存储)
- 游戏暂停/继续
- 下一预览方块显示
